# Telnet BYOD Readiness Check

This is a small Windows program Telnet asks homeworkers to run so we can confirm your computer is ready for Amazon Connect and working from home.

## Download

**[Download the latest version](https://github.com/TelnetServices/BYOD-Readiness-Check/releases/latest)**

Choose the `.exe` file on the release page (for example `TelnetReadinessCheck-v1.4.0.exe`).

Recent versions can update themselves automatically when a newer release is published here.

## How to run it

1. Download the `.exe` — no installation is required.
2. Double-click the file to open it.
3. Enter your name and email, then click **Start readiness check**.
4. Wait while it checks your PC and connection (this can take a few minutes).
5. When it finishes, save or email the report file to Telnet support as instructed.

You need:

- Windows 11 (64-bit)
- An internet connection
- Chrome or Microsoft Edge recommended (for the Connect and speed tests)

Your microphone may be requested during the Amazon Connect part of the check — that is normal.

## Privacy

This check runs on **your** PC. Telnet does not look at your browsing history, files, or passwords.

We only use your name and email so support can match the results to you.
